# BlueNumerals Dashboard Generator

A Streamlit app that generates a personalized dashboard demo from a public business website.

## Run Locally

```bash
pip install -r requirements.txt
streamlit run dash_app.py
```

## Deploy on Streamlit Community Cloud

Use these settings when creating the app:

- Repository: this project repository
- Branch: your deploy branch, usually `main`
- Main file path: `dash_app.py`

No secrets are required for the current version.
